# 07. 生徒詳細画面

**依存**: 06（出欠履歴の表示は 09 完了後に実データで確認）
**参照**: REQUIREMENTS.md §4.2・§4.4 / DESIGN.md §2（数字の見せ方）

## 目的

台帳情報と出欠履歴（月ごとの出席回数・請求額）を1画面で確認できる生徒詳細を実装する。

## Todo

- [x] 詳細画面 `/settings/students/[id]`: 台帳情報一式を表示する（氏名・ふりがな・所属コマ・単価・連絡先・生年月日・スマホ区分・緊急連絡先・メモ）
- [x] 出欠履歴セクション: 月ごとに出席回数 / 欠席回数 / 請求額（`SUM(unit_price_at_time) WHERE status='present'`）を新しい月から一覧表示する
- [x] 回数・金額は tabular-nums・右揃え。金額は `toLocaleString('ja-JP')`、「¥」は数字より一回り小さく muted
- [x] 編集画面（06）への動線を置く（テキストリンク）
- [x] 一覧（06）から詳細への遷移を接続する

## 完了条件

- 生徒1人の台帳情報と月別の出席回数・請求額が1画面で確認できる
