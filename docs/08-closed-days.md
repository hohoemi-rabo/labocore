# 08. 休講日管理

**依存**: 02, 04
**参照**: REQUIREMENTS.md §4.6 / DESIGN.md §5.5

## 目的

休講日（祝日・先生都合）の登録・削除を実装する。ホーム（09）とカレンダー（10）が参照する。

## Todo

- [ ] 一覧 `/settings/closed-days`: 日付+理由メモを日付順に表示する（過去分と今後分がわかる並び）
- [ ] 登録フォーム: 日付 + 理由メモ。Server Action + Zod で検証する
- [ ] 重複日付は `closed_date UNIQUE` 制約違反を捕捉して「既に登録済み」エラーを表示する
- [ ] 削除: 確認ダイアログ必須・ink 塗りボタン（物理削除でよい）
- [ ] 変更後は `revalidatePath` で関連画面（一覧・ホーム・カレンダー）を更新する

## 完了条件

- 休講日を登録・削除でき、登録済みの日はホーム/カレンダー実装時に参照できる
