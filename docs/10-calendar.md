# 10. 出欠カレンダー / 履歴修正

**依存**: 09（出欠トグル・記録 Action を再利用する）
**参照**: REQUIREMENTS.md §4.2 / DESIGN.md §5.4

## 目的

月表示カレンダーで記録状況を俯瞰し、任意の日（過去日含む）の出欠を確認・修正できるようにする。

## Todo

- [x] 出欠記録がある日に小さな Action Blue ドット、休講日は muted ドット+取り消し表現を表示する
- [x] 日付タップでその日の出欠パネルを開く（モバイル: 下からシート / PC: 右パネル）
- [x] パネル内: その日の曜日に該当するコマの在籍生徒+その日に記録があるイレギュラー生徒を一覧表示し、09 と同じ出欠トグルで修正できる（過去日の修正OK）
- [x] パネル内でも「生徒を追加」でその日の出席を追加できる
- [x] 出欠トグルコンポーネントと Server Action は 09 のものを共通化して使う（重複実装しない）。スナップショット規則も同一（記録時点の現在単価）
