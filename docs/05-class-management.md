# 05. コマ（クラス）管理

**依存**: 02, 04
**参照**: REQUIREMENTS.md §4.5 / DESIGN.md §5.5

## 目的

コマ（曜日・時間固定のクラス）の CRUD を実装する。生徒管理（06）の前提となるマスタ。

## Todo

- [x] 登録フォーム: コマ名 / 曜日（0=日〜6=土のセレクト）/ 開始・終了時間。追加ボタンは Action Blue ピル
- [x] 編集フォーム: 同項目。編集導線はテキストリンク（Action Blue）
